
# 📖 Interactive Python Dictionary

A smart, interactive command-line dictionary application built with Python. This tool allows users to search for word definitions from a JSON-based dataset and features an intelligent suggestion system for handling typos.

## ✨ Features

* **Intelligent Auto-Correction:** Uses the `difflib` library to suggest the closest matching word if the user provides an incorrect spelling.
* **Case Sensitivity Support:** Handles various input formats, including lowercase, uppercase (e.g., "USA"), and title case (e.g., "Paris").
* **Efficient Data Handling:** Utilizes a local `data.json` file for fast search queries.

## 🚀 Getting Started

### Prerequisites
Make sure you have Python installed. Check by running:
```bash
python --version

```
